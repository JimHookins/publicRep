This is HARMAN test task "Polygons".
The task is to develop a QML UI application, which draws a certain count of polygons. Polygons are located on the screen randomly.
If a user clicks on the polygon, this one disappears from the screen and other polygons appear. If there was no click event on the polygon, it has to disappear according to its lifespan time.
The number of a new appearing polygons equals to the number of disappearing polygons sides number.
The user can customize the initial polygons number, the range on which number of sides will be generated and the lifespan of every polygons.
